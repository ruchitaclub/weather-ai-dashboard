## ✨ Features

* 🌍 Real-time weather data
* 📅 Forecast visualization
* 📊 Interactive charts
* 🤖 AI-based temperature prediction
* 🌗 Dark / Light mode
* 🌍 Multi-city comparison

## 🛠 Tech Stack

* Python
* Streamlit
* Pandas
* Scikit-learn
* OpenWeather API
